// 與程式的初遇

我在求學階段就讀數位媒體設計系，研究所將畢業時初識程式語言。於美編設計的任職中，自主線上學習 HTML / CSS，程式語言能透過一行行的文字轉化構成視覺，排版的方式有別於平片設計相當靈活，我覺得很有趣。
 離職後，正巧遇到疫情爆發，因此決定計畫學習程式並規劃轉職，在此期間我先重新學習 HTML / CSS，接著學習 JQuery 與 Bootstradiv。

// 未來規劃

目標成為一名前端工程師，未來我規畫學習 JS 拓展自己的技能，並且持續練習既有技能。我的程式語言還不夠成熟，但我相信設計背景能夠為我加分成為我的優勢。過去工作中磨練我的溝通能力與團隊合作，期許能將技能搭配特質為團隊與公司帶來助益。
